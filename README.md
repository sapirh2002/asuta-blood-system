# מערכת ניהול לקיחת דם ועירוי — אסותא אשדוד

פרויקט גמר · מערכת אמיתית עם התחברות, מסד נתונים, פאנל אדמין וכל תיקוני אסותא (ללא AI).

## מה יש במערכת
- **התחברות אמיתית** (סיסמאות מוצפנות), הרשאות לפי תפקיד, ותוקף הרשאה עם התראת 30 יום.
- **מסד נתונים אמיתי** (SQLite): מטופלים, דגימות, הזמנות (כולל מרובות מרכיבים), עירויים, חתימות ו-Audit Trail.
- **תהליך מטופל מלא** לפי תיקוני אסותא: צליפת ידון חובה, אימות תלת-כיווני, סריקת מבחנה, דחיפויות מדויקות, חלונות קריטריונים (מוקרן/CMV/שטוף), 2 מזהים, סריקת מנה לפי סדר, מדדים חובה לפני עירוי, וחלון סיכום סיום עירוי.
- **פאנל אדמין**: ניהול משתמשים, מודול אינטגרציה (קמיליון/אוטופיוזן), דוחות ו-Audit, והגדרות מערכת (המטולוגים, קריטריונים, מחלקות).

## העלאה לאינטרנט (Render — חינם)
1. פתיחת חשבון חינם ב-render.com.
2. New → Web Service → חיבור מאגר Git (או Public Git Repo).
3. Render מזהה אוטומטית את `render.yaml` (Node 22, `npm install`, `node --experimental-sqlite server.js`).
4. Deploy → מתקבל קישור קבוע לשיתוף.
   * הערה: בתוכנית החינמית מסד הנתונים מתאפס בכל פריסה/הפעלה מחדש (מתאים להדגמה).

## מבנה
```
server.js            — שרת + כל ה-API (כולל אדמין)

public/              — index.html, styles.css, app.js
docs/אפיון-ותיקונים.md — ריכוז הדרישות והתיקונים
render.yaml          — הגדרות פריסה
```
